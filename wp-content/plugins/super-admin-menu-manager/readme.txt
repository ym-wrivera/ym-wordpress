== Description ==

This is a network/multisite only plugin.

This plugin was pulled from the Plugin Premium Package Manager for WP Networks, by Ron Rennick.

<b>Features:</b>

 * Simple multisite admin menu manager.

 * Manage all menus network wide.
 
 * Manage a subsite menu individually if needed.




== Installation ==



1. Extract `super-admin-menu-manager.zip` to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress Network Admin.

3. Go to the main site Admin-> Tools-> Manage Menus and toggle the menu items you wish to hide.



== Frequently Asked Questions ==



= How to use networkwide? =
Go to the main site Admin-> Tools-> Manage Menus and toggle the menu items you wish to hide.

= How to exclude a subsite menu in the network? =
If you wish to exclude a site from hiding menu items on that specific site, then log into that subsite as Super Admin and go to the site Admin-> Tools-> Manage Menus and toggle the "Show all menu items this site", then hit save. Saved menu settings will be specifically for that site.
